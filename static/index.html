Geoservice

Geoservice es un almacén para datos georeferenciados que proporciona, para cualquier catálogo de datos, los siguientes servicios:

  1. Data explorer

    Explora los datos de manera visual y construye consultas de manera sencilla.

  2. Mapa

    Navega por el mapa y busca resultados en un radio de acción

  3. API


    Dispone de una API REST que permite consultar los datos de manera sencilla y rápida e integrarlos en aplicaciones o exportarlos a otros formatos (json, csv, xls, kml).
    Permite:
    • Consultar por nombres de campos de un catálogo de datos
    • Consultar por dirección (geoaddress)
    • Ordenar los resultados
    • Paginar
    • Obtener los valores distintos de un campo (sql "distinct")
    • Y más...

    Geoservice API

  4. Layar

    Todos los catalágos de datos son expuestos directamente como capa de Layar
